## ADDED Requirements

### Requirement: Import action discoverable in Profile tab title

The Profile tab's title bar SHALL display the `I` (Import from file) shortcut hint alongside the existing `i` (Add from URL) hint, so users can discover the file import action.

#### Scenario: Title bar shows import hint

- **WHEN** the Profile tab is rendered and focused
- **THEN** the title bar or help area SHALL indicate that `I` triggers file import
